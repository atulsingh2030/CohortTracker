# WEBAPP TESTING SKILL

## When to use
Use this skill when the task is to run the local app, verify API responses, or test the product end to end on localhost.
Keywords: local app, localhost, interaction test, playwright.

## Inputs required
- App commands
- Expected routes or API endpoints

## Steps
1. Confirm which local services should be running.
2. Start only the services needed for the test.
3. Verify the page and API behavior together.
4. Report only what was directly observed.

## Output format
- Services started
- Endpoints checked
- UI states verified
- Issues or pass result

## Validation checklist
- [ ] Ports and commands were stated before execution
- [ ] API and UI were both checked when relevant
- [ ] Claims are runtime-verified
